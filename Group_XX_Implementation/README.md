# Implementation

This directory contains the main code implementation for the BDA IA2 project.

## Project Structure
- `main.ipynb`: Core implementation notebook containing logic, steps, and results documentation (using Markdown).
- `requirements.txt`: Python package dependencies.
- `Dataset/`: Folder for datasets.
- `Output/`: Folder for output results and graphs.
- `scripts/`: Helper python scripts.

## Setup Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Run `main.ipynb` in a Jupyter Notebook environment.
